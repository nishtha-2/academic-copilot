# DocuMentor

Documentor is an offline AI-powered academic assistant built using Retrieval-Augmented Generation (RAG).

## Features

- Hybrid Search (FAISS + BM25)
- CrossEncoder Re-ranking
- Parent-Child Chunking
- PDF Parsing
- OCR Support
- Diagram Retrieval
- Conversation Memory
- Semantic Memory
- Ollama (Qwen2.5)
- Streamlit Interface

## Tech Stack

- Python
- Streamlit
- FAISS
- BM25
- Sentence Transformers
- CrossEncoder
- Ollama
- PyMuPDF
- Tesseract OCR
